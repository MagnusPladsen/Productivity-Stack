# zellij

> Modern terminal multiplexer with a built-in layout system, floating panes, and WebAssembly plugin support.

**Platforms:** macOS, Linux

## Links
- [Official Website](https://zellij.dev/)
- [GitHub Repository](https://github.com/zellij-org/zellij) ⭐ 29.9k

## Install

### macOS
```bash
brew install zellij
```

### Linux
```bash
# Debian/Ubuntu
sudo apt install zellij

# Arch Linux
sudo pacman -S zellij

# Fedora
sudo dnf install zellij

# Cargo (requires Rust)
cargo install zellij
```

### Windows

Not officially supported. Use WSL2 with a Linux installation method above.

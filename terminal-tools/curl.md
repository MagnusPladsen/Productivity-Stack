# curl

> Command-line tool for transferring data with URLs — supports HTTP, HTTPS, FTP, and dozens more protocols.

**Platforms:** macOS, Linux, Windows

## Links
- [Official Website](https://curl.se/)

## Install

### macOS

curl ships with macOS by default. To get the latest version via Homebrew:

```bash
brew install curl
```

### Linux

**Debian/Ubuntu:**

```bash
sudo apt install curl
```

**Fedora:**

```bash
sudo dnf install curl
```

**Arch Linux:**

```bash
sudo pacman -S curl
```

### Windows

curl is bundled with Windows 10 (version 1803 and later) and Windows 11. To install or update via winget:

```bash
winget install curl.curl
```

Or download binaries from [curl.se/download.html](https://curl.se/download.html).

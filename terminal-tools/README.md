# Terminal Tools

Command-line utilities and shell tools to enhance your terminal workflow.

| Tool | Platforms | Open Source | Description |
|------|-----------|-------------|-------------|
| [Oh My Zsh](oh-my-zsh.md) | macOS, Linux | Yes | Community-driven framework for managing your Zsh configuration with plugins and themes |
| [Lazygit](lazygit.md) | macOS, Linux, Windows | Yes | Simple terminal UI for Git commands |
| [tmux](tmux.md) | macOS, Linux | Yes | Terminal multiplexer — split panes, create windows, and keep sessions alive after disconnect |
| [zoxide](zoxide.md) | macOS, Linux, Windows | Yes | A smarter cd command that learns your most-visited directories |
| [bat](bat.md) | macOS, Linux, Windows | Yes | A cat clone with syntax highlighting, line numbers, and Git integration |
| [eza](eza.md) | macOS, Linux | Yes | A modern, maintained replacement for ls with colors, icons, and Git status |
| [ripgrep](ripgrep.md) | macOS, Linux, Windows | Yes | Recursively searches directories for a regex pattern — faster than grep, respects .gitignore |
| [fd](fd.md) | macOS, Linux, Windows | Yes | A simple, fast, and user-friendly alternative to find |
| [htop](htop.md) | macOS, Linux | Yes | Interactive process viewer with color display, mouse support, and easy process management |
| [fzf](fzf.md) | macOS, Linux, Windows | Yes | A general-purpose command-line fuzzy finder for files, history, processes, and more |
| [Git](git.md) | macOS, Linux, Windows | Yes | Distributed version control system for tracking changes in source code |
| [curl](curl.md) | macOS, Linux, Windows | Yes | Command-line tool for transferring data with URLs — supports HTTP, HTTPS, FTP, and more |
| [tldr](tldr.md) | macOS, Linux, Windows | Yes | Simplified, community-driven man pages with practical examples |
| [jq](jq.md) | macOS, Linux, Windows | Yes | Lightweight and flexible command-line JSON processor |
| [ncdu](ncdu.md) | macOS, Linux | Yes | NCurses-based interactive disk usage analyzer |
| [asciinema](asciinema.md) | macOS, Linux | Yes | Record, share, and embed terminal sessions as lightweight text-based recordings |
| [taproom](taproom.md) | macOS | Yes | An interactive TUI for browsing, searching, and managing Homebrew packages |
| [fastfetch](fastfetch.md) | macOS, Linux, Windows | Yes | A fast, highly customizable system information tool in the style of neofetch |
| [figlet](figlet.md) | macOS, Linux | Yes | Renders text as large ASCII-art banners in the terminal |
| [thefuck](thefuck.md) | macOS, Linux | Yes | Magnificent app that auto-corrects your previous mistyped console command |
| [gh](gh.md) | macOS, Linux, Windows | Yes | GitHub's official CLI — manage pull requests, issues, and repos from the terminal |
| [yazi](yazi.md) | macOS, Linux, Windows | Yes | Blazing-fast terminal file manager written in Rust, with image previews and async I/O |
| [jolt](jolt.md) | macOS, Linux | Yes | Real-time battery and energy monitor TUI — shows power draw, process energy usage, and battery health |

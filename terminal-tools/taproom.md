# taproom

> An interactive TUI for browsing, searching, and managing Homebrew packages.

**Platforms:** macOS

## Links
- [GitHub Repository](https://github.com/hzqtc/taproom)

## Install

### macOS

```bash
brew install gromgit/brewtils/taproom
```

Or using [`eget`](https://github.com/zyedidia/eget):

```bash
eget hzqtc/taproom
```

## Alternatives
- [gh.md](gh.md) — GitHub CLI, an optional dependency taproom uses for release info

# tldr

> Simplified, community-driven man pages with practical examples.

**Platforms:** macOS, Linux, Windows

## Links
- [Official Website](https://tldr.sh/)
- [GitHub Repository](https://github.com/tldr-pages/tldr)

## Install

### macOS

```bash
brew install tlrc
```

### Linux

**Debian/Ubuntu:**

```bash
sudo apt install tldr
```

**Fedora:**

```bash
sudo dnf install tldr
```

**Arch Linux:**

```bash
sudo pacman -S tldr
```

Via Cargo (Rust client, all distros):

```bash
cargo install tlrc --locked
```

### Windows

```bash
winget install tldr-pages.tlrc
```

## Alternatives
- [curl.md](curl.md) — when you need the full man page depth rather than quick examples

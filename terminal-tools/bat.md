# bat

> A cat clone with syntax highlighting, line numbers, and Git integration.

**Platforms:** macOS, Linux, Windows

## Links
- [GitHub Repository](https://github.com/sharkdp/bat)

## Install

### macOS

```bash
brew install bat
```

### Linux

**Debian/Ubuntu:**

```bash
sudo apt install bat
```

> Note: on some Debian/Ubuntu systems the binary is installed as `batcat` due to a naming conflict. Create an alias: `alias bat='batcat'`

**Arch Linux:**

```bash
sudo pacman -S bat
```

**Fedora:**

```bash
sudo dnf install bat
```

### Windows

```bash
winget install sharkdp.bat
```

Or via Scoop:

```bash
scoop install bat
```

Or via Chocolatey:

```bash
choco install bat
```

# asciinema

> Record, share, and embed terminal sessions as lightweight text-based recordings.

**Platforms:** macOS, Linux

## Links
- [Official Website](https://asciinema.org/)
- [GitHub Repository](https://github.com/asciinema/asciinema) ⭐ 17.0k
- [Documentation](https://docs.asciinema.org/getting-started/)

## Install

### macOS

```bash
brew install asciinema
```

### Linux

**Debian/Ubuntu:**

```bash
sudo apt install asciinema
```

**Arch Linux:**

```bash
sudo pacman -S asciinema
```

Via Cargo (all distros, requires Rust 1.82+):

```bash
cargo install --locked --git https://github.com/asciinema/asciinema
```

# jolt

> Real-time battery and energy monitor TUI for laptops — shows power draw, process energy usage, and battery health.

**Platforms:** macOS, Linux

## Links
- [Official Website](https://getjolt.sh/)
- [GitHub Repository](https://github.com/jordond/jolt) ⭐ 443

## Install

### macOS

```bash
brew install jordond/tap/jolt
```

Or via the install script:

```bash
curl -fsSL https://getjolt.sh/install.sh | bash
```

### Linux

Via the install script:

```bash
curl -fsSL https://getjolt.sh/install.sh | bash
```

Or via Cargo:

```bash
cargo install jolt-tui
```

# htop

> Interactive process viewer with a color display, mouse support, and easy process management.

**Platforms:** macOS, Linux

## Links
- [Official Website](https://htop.dev/)
- [GitHub Repository](https://github.com/htop-dev/htop) ⭐ 7.9k
- [Downloads](https://htop.dev/downloads.html)

## Install

### macOS

```bash
brew install htop
```

### Linux

**Debian/Ubuntu:**

```bash
sudo apt install htop
```

**Fedora:**

```bash
sudo dnf install htop
```

**Arch Linux:**

```bash
sudo pacman -S htop
```

**RHEL/CentOS:**

```bash
sudo yum install htop
```

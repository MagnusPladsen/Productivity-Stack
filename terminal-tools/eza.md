# eza

> A modern, maintained replacement for ls with colors, icons, and Git status.

**Platforms:** macOS, Linux

## Links
- [Official Website](https://eza.rocks/)
- [GitHub Repository](https://github.com/eza-community/eza) ⭐ 20.5k
- [Installation Guide](https://github.com/eza-community/eza/blob/main/INSTALL.md)

## Install

### macOS

```bash
brew install eza
```

### Linux

**Arch Linux:**

```bash
sudo pacman -S eza
```

**Debian/Ubuntu (via binary release):**

```bash
wget -c https://github.com/eza-community/eza/releases/latest/download/eza_x86_64-unknown-linux-gnu.tar.gz -O - | tar xz
sudo mv eza /usr/local/bin/eza
```

**Fedora:**

```bash
sudo dnf install eza
```

**Void Linux:**

```bash
sudo xbps-install -S eza
```

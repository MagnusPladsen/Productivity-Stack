# ncdu

> NCurses-based interactive disk usage analyzer.

**Platforms:** macOS, Linux

## Links
- [Official Website](https://dev.yorhel.nl/ncdu)
- [GitHub Repository (unofficial fork)](https://github.com/rofl0r/ncdu)

## Install

### macOS

```bash
brew install ncdu
```

### Linux

**Debian/Ubuntu:**

```bash
sudo apt install ncdu
```

**Fedora:**

```bash
sudo dnf install ncdu
```

**Arch Linux:**

```bash
sudo pacman -S ncdu
```

**Alpine Linux:**

```bash
sudo apk add ncdu
```

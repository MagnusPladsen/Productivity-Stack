# gitui

> Fast terminal UI for git written in Rust — stage, commit, branch, and view diffs without leaving the terminal.

**Platforms:** macOS, Linux, Windows

## Links
- [Official Website](https://github.com/gitui-org/gitui)
- [GitHub Repository](https://github.com/gitui-org/gitui) ⭐ 21.5k

## Install

### macOS
```bash
brew install gitui
```

### Linux
```bash
# Cargo (requires Rust — recommended for latest version)
cargo install gitui --locked

# Arch Linux (AUR)
# Available via AUR helpers such as yay or paru
yay -S gitui
```

### Windows
```bash
winget install -e --id StephanDilly.gitui
```

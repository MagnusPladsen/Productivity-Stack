# tmux

> Terminal multiplexer — split panes, create windows, and keep sessions alive after disconnect.

**Platforms:** macOS, Linux

## Links
- [GitHub Repository](https://github.com/tmux/tmux)
- [Installation Wiki](https://github.com/tmux/tmux/wiki/Installing)

## Install

### macOS

```bash
brew install tmux
```

### Linux

**Debian/Ubuntu:**

```bash
sudo apt install tmux
```

**Fedora:**

```bash
sudo dnf install tmux
```

**Arch Linux:**

```bash
sudo pacman -S tmux
```

**RHEL/CentOS:**

```bash
sudo yum install tmux
```

## Alternatives
- [fzf.md](fzf.md) — pairs well with tmux for fuzzy-finding within sessions

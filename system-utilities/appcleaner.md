# AppCleaner

> Completely uninstall macOS apps along with all their associated files — preferences, caches, and support files.

**Platforms:** macOS

> **Note:** Freeware — free to use but not open source.

## Links
- [Official Website](https://freemacsoft.net/appcleaner/)

## Install

### macOS
```bash
brew install --cask appcleaner
```

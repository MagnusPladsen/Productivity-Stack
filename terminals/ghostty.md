# Ghostty

> Modern, fast terminal emulator using platform-native UI and GPU acceleration.

**Platforms:** macOS, Linux

## Links
- [Official Website](https://ghostty.org/)
- [GitHub Repository](https://github.com/ghostty-org/ghostty)

## Install

### macOS
```bash
brew install --cask ghostty
```

Or download the `.dmg` from [ghostty.org/docs/install/binary](https://ghostty.org/docs/install/binary).

### Linux

**Snap:**
```bash
snap install ghostty
```

**Nix:**
```bash
nix-env -iA nixpkgs.ghostty
```

For other distributions, see [ghostty.org/docs/install/binary](https://ghostty.org/docs/install/binary) for available packages or build from source.

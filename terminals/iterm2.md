# iTerm2

> Feature-rich macOS terminal replacement with split panes, search, autocomplete, and deep shell integration.

**Platforms:** macOS

## Links
- [Official Website](https://iterm2.com/)
- [GitHub Repository](https://github.com/gnachman/iTerm2) ⭐ 17.1k

## Install

### macOS
```bash
brew install --cask iterm2
```

Or download the `.zip` directly from [iterm2.com/downloads.html](https://iterm2.com/downloads.html).

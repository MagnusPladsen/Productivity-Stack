# Warp

> AI-powered terminal with agentic development features, block-based output, and cloud orchestration.

**Platforms:** macOS, Linux, Windows

## Links
- [Official Website](https://www.warp.dev/)
- [GitHub Repository](https://github.com/warpdotdev/Warp)
- [Documentation](https://docs.warp.dev/)

## Install

### macOS
```bash
brew install --cask warp
```

### Linux

Download the `.deb` or `.rpm` package from [warp.dev/download](https://www.warp.dev/download), then install:

**Debian/Ubuntu:**
```bash
sudo apt install ./<file>.deb
```

**RHEL/Fedora/CentOS:**
```bash
sudo dnf install ./<file>.rpm
```

### Windows
```bash
winget install Warp.Warp
```

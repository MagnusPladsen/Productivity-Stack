# cmux

> Ghostty-based macOS terminal with vertical tabs and notifications designed for running multiple AI coding agents simultaneously.

**Platforms:** macOS

## Links
- [Official Website](https://www.cmux.dev/)
- [GitHub Repository](https://github.com/manaflow-ai/cmux) ⭐ 5k
- [Documentation](https://term.cmux.dev/)

## Install

### macOS
```bash
brew tap manaflow-ai/cmux
brew install --cask cmux
```

Or download the `.dmg` from [GitHub Releases](https://github.com/manaflow-ai/cmux/releases).

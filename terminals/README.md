# Terminals

Terminal emulators for your development workflow.

| Tool | Platforms | Open Source | Description |
|------|-----------|-------------|-------------|
| [Kitty](kitty.md) | macOS, Linux | Yes | Fast, GPU-accelerated terminal emulator with a rich feature set and scriptable configuration |
| [iTerm2](iterm2.md) | macOS | Yes | Feature-rich macOS terminal replacement with split panes, search, and deep shell integration |
| [Warp](warp.md) | macOS, Linux, Windows | No | AI-powered terminal with agentic development features and block-based output |
| [Ghostty](ghostty.md) | macOS, Linux | Yes | Modern, fast terminal emulator using platform-native UI and GPU acceleration |
| [WezTerm](wezterm.md) | macOS, Linux, Windows | Yes | GPU-accelerated terminal emulator and multiplexer with Lua-based configuration |
| [cmux](cmux.md) | macOS | Yes | Ghostty-based terminal with vertical tabs and notifications for AI coding agents |

# Raycast

> A blazingly fast, totally extendable launcher for macOS — a Spotlight replacement with powerful extensions.

**Platforms:** macOS

## Links
- [Official Website](https://www.raycast.com/)
- [Extension Store](https://www.raycast.com/store)

## Install

### macOS

```bash
brew install --cask raycast
```

Or download the DMG from the [official website](https://www.raycast.com/).

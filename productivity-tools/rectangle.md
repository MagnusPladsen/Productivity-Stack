# Rectangle

> Move and resize windows on macOS using keyboard shortcuts and snap areas.

**Platforms:** macOS

## Links
- [Official Website](https://rectangleapp.com/)
- [GitHub Repository](https://github.com/rxhanson/Rectangle)

## Install

### macOS

```bash
brew install --cask rectangle
```

A paid pro version with additional features is also available:

```bash
brew install --cask rectangle-pro
```

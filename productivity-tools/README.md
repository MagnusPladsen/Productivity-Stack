## Window Management

Two distinct approaches are available on macOS:

- **Snap-based** — [Rectangle](rectangle.md): drag windows to screen edges or use keyboard shortcuts to snap to halves, thirds, quarters, etc.
- **Tiling** — [AeroSpace](aerospace.md): automatically tiles all windows; workspace switching is keyboard-driven, i3-style.

## Complementary Pairs

- [AeroSpace](aerospace.md) + [SketchyBar](sketchybar.md) + [JankyBorders](janky-borders.md) — popular combination for a fully custom tiling desktop
- [SketchyBar](sketchybar.md) + [JankyBorders](janky-borders.md) — both by FelixKratz, designed to work together

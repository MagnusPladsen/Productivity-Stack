# AeroSpace

> An i3-like tiling window manager for macOS with keyboard-driven workspace navigation.

**Platforms:** macOS

## Links
- [GitHub Repository](https://github.com/nikitabobko/AeroSpace)
- [Documentation](https://nikitabobko.github.io/AeroSpace/guide)

## Install

### macOS

```bash
brew install --cask nikitabobko/tap/aerospace
```

## Notes

- Does not require disabling System Integrity Protection (SIP)
- Configuration lives at `~/.aerospace.toml` or `~/.config/aerospace/aerospace.toml`
- Currently in public beta; breaking changes may occur before 1.0

## Alternatives
- [rectangle.md](rectangle.md) — simpler snap-based window management with GUI
- [sketchybar.md](sketchybar.md) — often paired with AeroSpace for a complete desktop setup

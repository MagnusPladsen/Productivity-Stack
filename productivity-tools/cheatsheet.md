# CheatSheet

> Hold Command (⌘) to display all available keyboard shortcuts for the current app.

**Platforms:** macOS

## Links
- [Official Website](https://www.mediaatelier.com/CheatSheet/)

## Install

### macOS

```bash
brew install --cask cheatsheet
```

> **Note:** CheatSheet has been discontinued and no longer works on macOS 14 Sonoma or later. Consider [KeyClu](https://sergii.tatarenkov.name/keyclu/support/) or [KeyCue](https://www.ergonis.com/products/keycue/) as actively maintained alternatives.

## Alternatives
- [raycast.md](raycast.md) — Raycast can surface shortcuts via extensions

# SketchyBar

> A highly customizable macOS status bar replacement.

**Platforms:** macOS

## Links
- [GitHub Repository](https://github.com/FelixKratz/SketchyBar) ⭐ 11.3k
- [Documentation](https://felixkratz.github.io/SketchyBar/)

## Install

### macOS

```bash
brew tap FelixKratz/formulae
brew install sketchybar
```

## Notes

- Requires **System Settings → Desktop & Dock → Displays have separate Spaces** to be enabled (this is the default)
- Configuration lives in `~/.config/sketchybar/`
- Pairs well with [janky-borders.md](janky-borders.md) — both are by FelixKratz

## Alternatives
- [aerospace.md](aerospace.md) — commonly used alongside SketchyBar for a tiling + custom bar setup
- [janky-borders.md](janky-borders.md) — complementary tool for window border decoration

# Shottr

> Instant screenshot tool for macOS with built-in OCR, pixel ruler, color picker, annotation tools, and scrolling capture.

**Platforms:** macOS

**Pricing:** 100% free (freeware). Requires macOS 10.15 or later.

## Links
- [Official Website](https://shottr.cc)

## Install

### macOS
```bash
brew install --cask shottr
```

Or download the `.dmg` directly from [shottr.cc](https://shottr.cc).

## Alternatives

For cross-platform screenshot and annotation, see [Flameshot](../screenshot-tools/flameshot.md).

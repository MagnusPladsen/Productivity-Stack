# AltTab

> Windows-style alt-tab window switching for macOS.

**Platforms:** macOS

## Links
- [Official Website](https://alt-tab-macos.netlify.app/)
- [GitHub Repository](https://github.com/lwouis/alt-tab-macos)

## Install

### macOS

```bash
brew install --cask alt-tab
```

## Notes

After installing, grant the required permissions in **System Settings → Privacy & Security → Accessibility**.

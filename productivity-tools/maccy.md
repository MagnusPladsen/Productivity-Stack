# Maccy

> Lightweight, keyboard-driven clipboard manager for macOS — keeps a searchable history of everything you copy.

**Platforms:** macOS

## Links
- [Official Website](https://maccy.app)
- [GitHub Repository](https://github.com/p0deje/Maccy)

## Install

### macOS
```bash
brew install --cask maccy
```

Or download the `.dmg` from the [GitHub Releases page](https://github.com/p0deje/Maccy/releases).

Also available on the [Mac App Store](https://apps.apple.com/app/maccy/id1527619437).

## Usage

Press `Shift + Cmd + C` (or click the menu bar icon) to open the clipboard history. Type to search, press `Enter` to paste.

## Alternatives

No direct alternatives in this folder. For cross-platform clipboard management, see system-level tools in your package manager.

# Espanso

> A privacy-first, cross-platform text expander written in Rust.

**Platforms:** macOS, Linux, Windows

## Links
- [Official Website](https://espanso.org/)
- [GitHub Repository](https://github.com/espanso/espanso) ⭐ 13.4k
- [Documentation](https://espanso.org/docs/get-started/)

## Install

### macOS

```bash
brew install espanso
```

### Linux

Download the `.deb` package for Debian/Ubuntu:

```bash
wget https://github.com/espanso/espanso/releases/latest/download/espanso-debian-amd64.deb
sudo apt install ./espanso-debian-amd64.deb
```

For other distributions, see the [Linux install guide](https://espanso.org/docs/install/linux).

### Windows

```bash
winget install --id Espanso.Espanso -e
```

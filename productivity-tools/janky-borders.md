# JankyBorders

> A lightweight window border system for macOS that highlights the focused window.

**Platforms:** macOS

## Links
- [GitHub Repository](https://github.com/FelixKratz/JankyBorders)

## Install

### macOS

```bash
brew tap FelixKratz/formulae
brew install borders
```

## Notes

- Requires macOS 14.0 or later
- Start with `borders &` to see borders around all windows
- Configuration via `~/.config/borders/bordersrc` or CLI flags
- See available options with `man borders`
- Pairs well with [sketchybar.md](sketchybar.md) — both are by FelixKratz

## Alternatives
- [sketchybar.md](sketchybar.md) — complementary tool for a fully customized macOS desktop

# Ollama

> Run large language models locally with a single command.

**Platforms:** macOS, Linux, Windows

## Links
- [Official Website](https://ollama.com)
- [GitHub Repository](https://github.com/ollama/ollama)

## Install

### macOS
```bash
brew install ollama
```
Or download the `.dmg` from [ollama.com/download/mac](https://ollama.com/download/mac) and drag to Applications.

### Linux
```bash
curl -fsSL https://ollama.com/install.sh | sh
```

### Windows
Download the `.exe` installer from [ollama.com/download](https://ollama.com/download) and run it.

## Usage
```bash
ollama run llama3.2
```

## Pricing
100% free and open source. No usage limits, no API keys required.

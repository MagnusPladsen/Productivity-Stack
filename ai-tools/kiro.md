# Kiro

> Amazon's agentic AI IDE with spec-driven development — turns prompts into specs, then into working code.

**Platforms:** macOS, Linux, Windows

## Links
- [Official Website](https://kiro.dev)
- [GitHub Repository](https://github.com/kirodotdev/Kiro) ⭐ 3.2k
- [Downloads](https://kiro.dev/downloads/)

## Install

### macOS / Linux / Windows (IDE)
Download the installer for your platform from [kiro.dev/downloads](https://kiro.dev/downloads/) and run it.

### macOS / Linux (CLI)
```bash
curl -fsSL https://cli.kiro.dev/install | bash
```

Sign in with GitHub, Google, AWS Builder ID, or AWS IAM Identity Center — no AWS account required.

## Pricing
Free during preview with no hard limits on usage. After the preview period ends, a free tier with 50 interactions/month will be available. Paid plans are planned at $19/month (1,000 interactions) and $39/month (3,000 interactions).

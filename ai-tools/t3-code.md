# T3 Code

> Desktop app and web GUI for managing AI coding agents with multi-repo and worktree support.

**Platforms:** macOS, Linux, Windows

## Links
- [Official Website](https://t3.codes/)
- [GitHub Repository](https://github.com/pingdotgg/t3code) ⭐ 6.5k

## Install

### macOS / Linux / Windows (npx)

```bash
npx t3-code@latest
```

### macOS

```bash
brew install --cask t3-code
```

# Antigravity

> Google's AI-first IDE powered by Gemini 3, with agentic development and multi-model support.

**Platforms:** macOS, Linux, Windows

## Links
- [Official Website](https://antigravity.google)
- [Announcement](https://developers.googleblog.com/build-with-google-antigravity-our-new-agentic-development-platform/)

## Install

### macOS / Linux / Windows
Download the installer for your platform from [antigravity.google/download](https://antigravity.google/download) and run it.

**System requirements:**
- Windows 10 or later (x64 or ARM64)
- macOS 12 (Monterey) or later
- Linux: glibc 2.28+ (Ubuntu 20.04+, Debian 10+, Fedora 36+, RHEL 8+)

## Pricing
Free during public preview — all features available at no cost with generous Gemini 3 Pro rate limits. Pricing after preview has not been announced.

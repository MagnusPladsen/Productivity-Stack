# Goose

> Open-source on-machine AI agent by Block that can install, execute, edit, and test with any LLM.

**Platforms:** macOS, Linux, Windows (via WSL or Git Bash)

## Links
- [GitHub Repository](https://github.com/block/goose)
- [Documentation](https://block.github.io/goose)

## Install

### macOS
```bash
curl -fsSL https://github.com/block/goose/releases/download/stable/download_cli.sh | bash
```

Or install the desktop app via Homebrew:
```bash
brew install --cask goose
```

### Linux
```bash
curl -fsSL https://github.com/block/goose/releases/download/stable/download_cli.sh | bash
```

Desktop packages (DEB, RPM, Flatpak) are available on the [releases page](https://github.com/block/goose/releases).

### Windows
Windows is supported via WSL (Windows Subsystem for Linux) or Git Bash. Run the Linux install script inside your WSL terminal. A native Windows desktop installer is also available on the [releases page](https://github.com/block/goose/releases).

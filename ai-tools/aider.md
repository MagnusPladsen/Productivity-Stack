# Aider

> AI pair programming in the terminal with a git-native workflow.

**Platforms:** macOS, Linux, Windows

## Links
- [Official Website](https://aider.chat)
- [GitHub Repository](https://github.com/Aider-AI/aider) ⭐ 41.8k

## Install

### macOS
```bash
brew install aider
```

### Linux / Windows
```bash
pip install aider-chat
```

Or using `pipx` for isolated installation:
```bash
pipx install aider-chat
```

Or using `uv`:
```bash
uv tool install --force --python python3.12 --with pip aider-chat@latest
```

## Pricing
Free and open source (BYOK — Bring Your Own Key). You supply your own API key from providers such as Anthropic, OpenAI, or use local models via Ollama. No subscription required for Aider itself.

# Gemini CLI

> Google's open-source terminal AI agent powered by Gemini models.

**Platforms:** macOS, Linux, Windows

## Links
- [GitHub Repository](https://github.com/google-gemini/gemini-cli)

## Install

### macOS / Linux / Windows
```bash
npm install -g @google/gemini-cli
```

Or run without installing:
```bash
npx @google/gemini-cli
```

**Requirements:** Node.js 20 or higher.

## Pricing
Free tier: 1,000 requests/day and 60 requests/minute with a personal Google account. No credit card required. Paid tiers available via Google AI Studio or Vertex AI for higher limits.

# Roo Code

> Feature-rich Cline fork for VS Code with sub-agent orchestration, multiple modes, and enhanced context management.

**Platforms:** macOS, Linux, Windows (VS Code extension)

## Links
- [Official Website](https://roocode.com)
- [GitHub Repository](https://github.com/RooCodeInc/Roo-Code)
- [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=RooVeterinaryInc.roo-cline)

## Install

### macOS / Linux / Windows
Install via VS Code Extensions (Ctrl/Cmd+Shift+X), search for **"Roo Code"**, and click Install.

Or from the terminal:
```bash
code --install-extension RooVeterinaryInc.roo-cline
```

## Pricing
Free and open source (BYOK — Bring Your Own Key). You supply your own API key from any supported provider (Anthropic, OpenAI, Google, etc.) and pay per token directly to the provider. No subscription required for the extension itself.

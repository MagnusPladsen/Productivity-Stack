# Continue

> Open-source Copilot replacement for VS Code and JetBrains with chat, autocomplete, and edit.

**Platforms:** macOS, Linux, Windows

## Links
- [Official Website](https://continue.dev)
- [GitHub Repository](https://github.com/continuedev/continue)
- [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=Continue.continue)
- [JetBrains Plugin](https://plugins.jetbrains.com/plugin/22707-continue)

## Install

### macOS / Linux / Windows (VS Code)
Install via VS Code Extensions (Ctrl/Cmd+Shift+X), search for **"Continue"**, and click Install.

Or from the terminal:
```bash
code --install-extension Continue.continue
```

### JetBrains IDEs
Open Settings (Cmd/Ctrl+Shift+,) → Plugins → search **"Continue"** → Install.

## Pricing
Free and open source (BYOK — Bring Your Own Key). Connect your own API keys from Anthropic, OpenAI, Google, or local models via Ollama. No subscription required for the extension itself.

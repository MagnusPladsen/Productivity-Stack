# Cursor

> AI code editor with deep codebase understanding, chat, and inline editing.

**Platforms:** macOS, Linux, Windows

## Links
- [Official Website](https://cursor.com)

## Install

### macOS
Download the `.dmg` from [cursor.com/download](https://cursor.com/download) and drag to Applications.

### Linux
Download the AppImage from [cursor.com/download](https://cursor.com/download), then:
```bash
chmod +x cursor-*.AppImage
./cursor-*.AppImage
```

A `.deb` package for Debian/Ubuntu is also available on the download page.

### Windows
Download the `.exe` installer from [cursor.com/download](https://cursor.com/download) and run it.

## Pricing
Free tier includes: 2,000 code completions/month and 50 premium model requests/month. Paid Pro plan at $20/month for unlimited completions and 500 premium requests/month.

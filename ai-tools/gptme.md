# gptme

> Minimal terminal AI agent with shell, browser, and code execution access.

**Platforms:** macOS, Linux

## Links
- [Official Website](https://gptme.org)
- [GitHub Repository](https://github.com/gptme/gptme)

## Install

### macOS / Linux
```bash
pipx install gptme
```

Or with pip:
```bash
pip install gptme
```

**Requirements:** Python 3.10 or higher.

## Pricing
Free and open source (BYOK — Bring Your Own Key). You supply your own API key from providers such as Anthropic or OpenAI. Can also run against local models via Ollama. No subscription required for gptme itself.

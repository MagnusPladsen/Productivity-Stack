# Cline

> Autonomous AI coding agent for VS Code that can read, write, and execute code.

**Platforms:** macOS, Linux, Windows (VS Code extension)

## Links
- [Official Website](https://cline.bot)
- [GitHub Repository](https://github.com/cline/cline) ⭐ 58.9k
- [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=saoudrizwan.claude-dev)

## Install

### macOS / Linux / Windows
Install via VS Code Extensions (Ctrl/Cmd+Shift+X), search for **"Cline"**, and click Install.

Or install from the terminal:
```bash
code --install-extension saoudrizwan.claude-dev
```

## Pricing
Free and open source (BYOK — Bring Your Own Key). You supply your own API key from any supported provider (Anthropic, OpenAI, Google, etc.) and pay per token directly to the provider. No subscription required for the extension itself.

# Monitoring

Tools for observing infrastructure health, building dashboards, and load testing.

| Tool | Platforms | Open Source | Description |
|------|-----------|-------------|-------------|
| [Netdata](netdata.md) | macOS, Linux | Yes | Real-time infra monitoring at 1-second resolution (agent free; cloud has free tier) |
| [Grafana](grafana.md) | macOS, Linux, Windows | Yes | Dashboard and visualization platform (self-hosted free; cloud has free tier) |
| [k6](k6.md) | macOS, Linux, Windows | Yes | Load testing written as JavaScript |

# Docker

> Container platform for building, sharing, and running containerized applications.

**Platforms:** macOS, Linux, Windows

## Links
- [Official Website](https://www.docker.com/)
- [GitHub Repository](https://github.com/docker/docker-ce)
- [Documentation](https://docs.docker.com/)

## Install

### macOS

```bash
brew install --cask docker-desktop
```

Or download the installer from the [official downloads page](https://www.docker.com/products/docker-desktop/).

### Linux

Docker Engine can be installed directly on Linux (no GUI). Official install script for Ubuntu/Debian:

```bash
curl -fsSL https://get.docker.com | sh
```

For Docker Desktop on Linux, download the `.deb` or `.rpm` package from the [official downloads page](https://www.docker.com/products/docker-desktop/).

### Windows

```bash
winget install Docker.DockerDesktop
```

Requires WSL2 (Windows Subsystem for Linux 2) to be enabled first.

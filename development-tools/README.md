# Development Tools

API clients, database managers, containers, debuggers, and CLI utilities for development.

| Tool | Platforms | Open Source | Description |
|------|-----------|-------------|-------------|
| [Beekeeper Studio](beekeeper-studio.md) | macOS, Linux, Windows | Yes (Community) | Modern SQL client for MySQL, Postgres, SQLite, SQL Server, and more |
| [Bruno](bruno.md) | macOS, Linux, Windows | Yes | Git-native API client that stores collections as plain files |
| [DBeaver](dbeaver.md) | macOS, Linux, Windows | Yes (Community) | Universal database management tool with SQL editor and ER diagrams |
| [Docker](docker.md) | macOS, Linux, Windows | Yes (Engine) | Container platform for building and running containerized applications |
| [Hoppscotch](hoppscotch.md) | macOS, Linux, Windows | Yes | Open-source web-based API client for REST, GraphQL, WebSocket, and more |
| [HTTPie](httpie.md) | macOS, Linux, Windows | Yes (CLI) | Human-friendly HTTP client for the CLI and desktop |
| [Insomnia](insomnia.md) | macOS, Linux, Windows | Yes | API client for REST, GraphQL, WebSockets, SSE, and gRPC |
| [Portainer](portainer.md) | macOS, Linux, Windows | Yes (CE) | Web UI for managing Docker and Kubernetes environments |
| [Postman](postman.md) | macOS, Linux, Windows | No | Full-featured API development environment |
| [shadcn CLI](shadcn-cli.md) | macOS, Linux, Windows | Yes | CLI for adding UI components to React/Next.js projects |
| [RAD Debugger](rad-debugger.md) | Windows | Yes | Native, multi-process graphical debugger (alpha) |
| [RenderDoc](renderdoc.md) | Windows, Linux | Yes | Graphics frame debugger for Vulkan, D3D, and OpenGL |

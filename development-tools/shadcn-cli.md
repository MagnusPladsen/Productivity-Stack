# shadcn CLI

> CLI for adding accessible, customizable UI components to React and Next.js projects.

**Platforms:** macOS, Linux, Windows

## Links
- [Official Website](https://ui.shadcn.com/)
- [GitHub Repository](https://github.com/shadcn-ui/ui)
- [Documentation](https://ui.shadcn.com/docs)
- [CLI Reference](https://ui.shadcn.com/docs/cli)

## Install

The CLI runs via a package manager — no global install required.

### Initialize a new project

```bash
# npm
npx shadcn@latest init

# bun
bunx shadcn@latest init

# pnpm
pnpm dlx shadcn@latest init

# yarn
yarn dlx shadcn@latest init
```

### Add components

```bash
npx shadcn@latest add button
npx shadcn@latest add card input label
```

> Note: The package was renamed from `shadcn-ui` to `shadcn`. Use `shadcn@latest`, not `shadcn-ui@latest`.

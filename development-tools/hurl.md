# Hurl

> Plain-text HTTP request runner for testing REST APIs, HTML responses, and GraphQL — runs in CI/CD pipelines without a GUI.

**Platforms:** macOS, Linux, Windows

## Links
- [Official Website](https://hurl.dev)
- [GitHub Repository](https://github.com/Orange-OpenSource/hurl)
- [Documentation](https://hurl.dev/docs/installation.html)

## Install

### macOS
```bash
brew install hurl
```

### Linux

**Debian/Ubuntu (via PPA):**
```bash
sudo add-apt-repository ppa:lepapareil/hurl
sudo apt update && sudo apt install hurl
```

Or download a `.deb` directly from the [GitHub Releases page](https://github.com/Orange-OpenSource/hurl/releases).

**Cargo (any distro):**
```bash
cargo install hurl
```

### Windows
```bash
winget install hurl
```

Or with Scoop:
```bash
scoop install hurl
```

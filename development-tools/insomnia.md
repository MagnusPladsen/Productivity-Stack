# Insomnia

> Open-source API client for REST, GraphQL, WebSockets, SSE, and gRPC.

**Platforms:** macOS, Linux, Windows

## Links
- [Official Website](https://insomnia.rest/)
- [GitHub Repository](https://github.com/Kong/insomnia) ⭐ 38.0k
- [Documentation](https://docs.insomnia.rest/)

## Install

### macOS

```bash
brew install --cask insomnia
```

### Linux

```bash
snap install insomnia
```

Or download an AppImage, `.deb`, or `.rpm` from the [downloads page](https://insomnia.rest/download).

### Windows

```bash
winget install Insomnia.Insomnia
```

Or download the installer from the [downloads page](https://insomnia.rest/download).

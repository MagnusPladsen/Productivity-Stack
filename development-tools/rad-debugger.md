# RAD Debugger

> Native, user-mode, multi-process graphical debugger for Windows (Linux support planned).

**Platforms:** Windows

## Links
- [Official Website](https://www.radgametools.com/debug.htm)
- [GitHub Repository](https://github.com/EpicGamesExt/raddebugger)
- [Releases](https://github.com/EpicGamesExt/raddebugger/releases)

## Install

### Windows

Download the latest pre-built binary from the [GitHub Releases page](https://github.com/EpicGamesExt/raddebugger/releases). Extract the archive and run `rad_debugger.exe` directly — no installer required.

> Currently supports Windows x64 debugging with PDB debug info. The project is in alpha. Linux support is planned for a future release.

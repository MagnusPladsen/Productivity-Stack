# Beekeeper Studio

> Modern, open-source SQL client for MySQL, Postgres, SQLite, SQL Server, and more.

**Platforms:** macOS, Linux, Windows

**Pricing:** Community Edition is 100% free and open source. Ultimate Edition adds features like table editing, query magician, and priority support — paid license required.

## Install

### macOS

```bash
brew install --cask beekeeper-studio
```

### Linux

**Snap (recommended):**

```bash
sudo snap install beekeeper-studio
```

Note: To use SSH tunneling with the snap version, enable key access:

```bash
sudo snap connect beekeeper-studio:ssh-keys :ssh-keys
```

**AppImage:** Download from [beekeeperstudio.io/get](https://www.beekeeperstudio.io/get) and mark as executable:

```bash
chmod +x Beekeeper-Studio-*.AppImage
./Beekeeper-Studio-*.AppImage
```

### Windows

Download the installer from [beekeeperstudio.io/get](https://www.beekeeperstudio.io/get) and run it.

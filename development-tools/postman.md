# Postman

> API development environment for designing, testing, and documenting APIs.

**Platforms:** macOS, Linux, Windows

## Links
- [Official Website](https://www.postman.com/)
- [Documentation](https://learning.postman.com/docs/)
- [Downloads](https://www.postman.com/downloads/)

## Install

### macOS

```bash
brew install --cask postman
```

### Linux

```bash
snap install postman
```

Or download a `.tar.gz` from the [downloads page](https://www.postman.com/downloads/).

### Windows

```bash
winget install Postman.Postman
```

## Alternatives
- [bruno.md](bruno.md) — git-native, offline-first API client

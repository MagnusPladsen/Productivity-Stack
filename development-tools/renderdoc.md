# RenderDoc

> Stand-alone graphics frame debugger for Vulkan, D3D11, D3D12, and OpenGL/OpenGL ES.

**Platforms:** Windows, Linux

## Links
- [Official Website](https://renderdoc.org/)
- [GitHub Repository](https://github.com/baldurk/renderdoc) ⭐ 10.5k
- [Documentation](https://renderdoc.org/docs/)
- [Downloads](https://renderdoc.org/builds)

## Install

### Windows

```bash
winget install -e --id BaldurKarlsson.RenderDoc
```

Or download the installer from the [builds page](https://renderdoc.org/builds).

### Linux

Download the precompiled binary tarball for x86_64 from the [builds page](https://renderdoc.org/builds), or build from source via the GitHub repository.

Some distributions also package RenderDoc in their official repos:

```bash
# Arch Linux
sudo pacman -S renderdoc
```

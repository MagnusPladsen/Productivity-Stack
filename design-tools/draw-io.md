# draw.io

> Free, open-source diagramming tool for flowcharts, network diagrams, UML, and more. Also known as diagrams.net.

**Platforms:** macOS, Linux, Windows (also available as a web app)

## Links
- [Official Website](https://www.drawio.com/)
- [GitHub Repository](https://github.com/jgraph/drawio-desktop)
- [Web App](https://app.diagrams.net/)

## Install

### macOS
```bash
brew install --cask drawio
```

### Linux
Download the `.AppImage` or `.deb` from the [GitHub Releases page](https://github.com/jgraph/drawio-desktop/releases).

### Windows
```bash
winget install -e --id JGraph.Draw
```

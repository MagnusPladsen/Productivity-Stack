# Design Tools

Design, prototyping, and diagramming applications.

| Tool | Platforms | Open Source | Description |
|------|-----------|-------------|-------------|
| [Penpot](penpot.md) | Web / Self-hosted | Yes | Open-source design and prototyping tool built on web standards |
| [Modulz](modulz.md) | Web | Yes (archived) | Visual design system editor — discontinued in 2022; succeeded by Radix UI |
| [draw.io](draw-io.md) | macOS, Linux, Windows, Web | Yes | Free diagramming tool for flowcharts, UML, and network diagrams |

# Design Tools

Design, prototyping, and diagramming applications.

| Tool | Platforms | Open Source | Description |
|------|-----------|-------------|-------------|
| [Penpot](penpot.md) ⭐ 44.8k | Web / Self-hosted | Yes | Open-source design and prototyping tool built on web standards |
| [Modulz](modulz.md) | Web | Yes (archived) | Visual design system editor — discontinued in 2022; succeeded by Radix UI |
| [draw.io](draw-io.md) ⭐ 59.8k | macOS, Linux, Windows, Web | Yes | Free diagramming tool for flowcharts, UML, and network diagrams |
| [Excalidraw](excalidraw.md) ⭐ 118.6k | Web | Yes | Hand-drawn style whiteboard for diagrams and wireframes; Excalidraw+ is paid |
| [Mermaid](mermaid.md) ⭐ 86.7k | Web / CLI | Yes | Diagram-as-code natively supported in GitHub, GitLab, and Obsidian |
| [D2](d2.md) ⭐ 23.2k | macOS, Linux, Windows | Yes | Modern diagram scripting language with a watch mode and VS Code extension |

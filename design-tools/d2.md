# D2

> Modern diagram scripting language that turns plain text into polished diagrams. Supports flowcharts, entity-relationship diagrams, sequence diagrams, and more.

**Platforms:** macOS, Linux, Windows

## Links
- [Official Website](https://d2lang.com)
- [GitHub Repository](https://github.com/terrastruct/d2)
- [Playground](https://play.d2lang.com)

## Install

### macOS
```bash
brew install d2
```

### Linux

Download a pre-built binary from the [GitHub Releases page](https://github.com/terrastruct/d2/releases), or use the install script:

```bash
curl -fsSL https://d2lang.com/install.sh | sh -
```

### Windows
```bash
winget install Terrastruct.D2
```

Or download the `.msi` installer from the [GitHub Releases page](https://github.com/terrastruct/d2/releases).

## Usage

```bash
d2 input.d2 output.svg
```

Run with `--watch` to auto-render on save:

```bash
d2 --watch input.d2 output.svg
```

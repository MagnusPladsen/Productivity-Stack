# Mermaid

> Diagram-as-code library that renders flowcharts, sequence diagrams, Gantt charts, and more from simple text syntax. Natively supported in GitHub, GitLab, and Obsidian.

**Platforms:** Web / CLI (macOS, Linux, Windows)

## Links
- [Official Website](https://mermaid.js.org)
- [GitHub Repository](https://github.com/mermaid-js/mermaid) ⭐ 86.7k
- [Live Editor](https://mermaid.live)

## Use

Paste Mermaid syntax into [mermaid.live](https://mermaid.live) for instant rendering — no install needed.

In GitHub Markdown, wrap diagrams in a fenced code block with `mermaid` as the language.

## Install CLI

### macOS
```bash
brew install mermaid-cli
```

### Linux / Windows

**npm (global):**
```bash
npm install -g @mermaid-js/mermaid-cli
```

Then run diagrams with:
```bash
mmdc -i input.mmd -o output.svg
```

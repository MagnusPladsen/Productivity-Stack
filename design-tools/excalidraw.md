# Excalidraw

> Virtual whiteboard with a hand-drawn aesthetic for sketching diagrams, wireframes, and flowcharts — works entirely in the browser.

**Platforms:** Web (self-hostable via Docker)

**Pricing:** 100% free and open source. Excalidraw+ is a paid cloud service ($7/month) with persistent storage and real-time collaboration.

## Links
- [Official Website](https://excalidraw.com)
- [GitHub Repository](https://github.com/excalidraw/excalidraw)
- [Documentation](https://docs.excalidraw.com)

## Use

Open [excalidraw.com](https://excalidraw.com) in any browser — no install required.

## Self-host with Docker

```bash
docker run -p 3000:80 excalidraw/excalidraw:latest
```

Then open `http://localhost:3000`.

## Embed as a React component

```bash
npm install @excalidraw/excalidraw
```

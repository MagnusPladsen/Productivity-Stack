# Firefox

> Mozilla's independent Gecko-based browser with multi-account containers, strong privacy defaults, and a large extension ecosystem.

**Platforms:** macOS, Linux, Windows

## Links
- [Official Website](https://mozilla.org/firefox)
- [Download Page](https://www.mozilla.org/firefox/new/)

## Install

### macOS
```bash
brew install --cask firefox
```

### Linux

Firefox is available in most distribution repositories:

**Debian/Ubuntu:**
```bash
sudo apt install firefox
```

**Arch Linux:**
```bash
sudo pacman -S firefox
```

**Flatpak (all distros):**
```bash
flatpak install flathub org.mozilla.firefox
```

### Windows
```bash
winget install Mozilla.Firefox
```

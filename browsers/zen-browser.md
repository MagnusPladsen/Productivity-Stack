# Zen Browser

> A minimalist, privacy-focused browser built on Firefox with a calm, distraction-free interface.

**Platforms:** macOS, Linux, Windows

## Links
- [Official Website](https://zen-browser.app/)
- [GitHub Repository](https://github.com/zen-browser/desktop) ⭐ 40.6k
- [Downloads](https://zen-browser.app/download/)

## Install

### macOS

```bash
brew install --cask zen-browser
```

### Linux

Install via Flatpak:

```bash
flatpak install flathub io.github.zen_browser.zen
```

Or download the AppImage from the [official downloads page](https://zen-browser.app/download/).

### Windows

```bash
winget install --id Zen-Team.Zen-Browser.Optimized
```

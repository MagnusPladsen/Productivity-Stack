# Browsers

Alternative web browsers focused on privacy, performance, or minimalism.

## Comparison

| Tool | Engine | Platforms | Free | Focus |
|------|--------|-----------|------|-------|
| [Zen Browser](zen-browser.md) ⭐ 40.6k | Firefox (Gecko) | macOS, Linux, Windows | Yes | Minimalism, privacy, distraction-free browsing |
| [Firefox](firefox.md) | Gecko | macOS, Linux, Windows | Yes | Independent browser with multi-account containers and strong extension ecosystem |
| [Brave](brave.md) ⭐ 21.9k | Chromium | macOS, Linux, Windows | Yes | Built-in ad/tracker blocking; optional paid VPN |
| [Floorp](floorp.md) ⭐ 8.0k | Firefox (Gecko) | macOS, Linux, Windows | Yes | Firefox fork with vertical tabs and split-view browsing |

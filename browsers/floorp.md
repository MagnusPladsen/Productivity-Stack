# Floorp

> Firefox fork built in Japan with vertical tabs, split-view browsing, deep UI customization, and built-in privacy features.

**Platforms:** macOS, Linux, Windows

## Links
- [Official Website](https://floorp.app)
- [GitHub Repository](https://github.com/Floorp-Projects/Floorp) ⭐ 8.0k
- [Documentation](https://docs.floorp.app)

## Install

### macOS
```bash
brew install --cask floorp
```

### Linux

**Flatpak (all distros):**
```bash
flatpak install https://dl.flathub.org/repo/appstream/one.ablaze.floorp.flatpakref
```

Or add the official apt repository for Debian/Ubuntu — see [installation instructions](https://docs.floorp.app/docs/installation/).

### Windows
```bash
winget install -e --id Ablaze.Floorp
```

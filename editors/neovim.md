# Neovim

> Extensible Vim-based text editor with a focus on extensibility, usability, and modern plugin APIs.

**Platforms:** macOS, Linux, Windows

## Links
- [Official Website](https://neovim.io/)
- [GitHub Repository](https://github.com/neovim/neovim) ⭐ 97.2k
- [Install Guide](https://github.com/neovim/neovim/blob/master/INSTALL.md)

## Install

### macOS
```bash
brew install neovim
```

### Linux
```bash
# Ubuntu / Debian (via snap)
sudo snap install nvim --classic

# Arch Linux
sudo pacman -S neovim

# Fedora
sudo dnf install neovim
```

### Windows
```bash
winget install Neovim.Neovim
```

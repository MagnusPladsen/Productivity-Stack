# Neovide

> No-nonsense Neovim GUI client written in Rust, with smooth animations and a native feel.

**Platforms:** macOS, Linux, Windows

## Links
- [Official Website](https://neovide.dev/)
- [GitHub Repository](https://github.com/neovide/neovide) ⭐ 14.8k
- [Installation Guide](https://neovide.dev/installation.html)

## Install

Requires [Neovim](neovim.md) 0.10 or newer to be installed and on your `PATH`.

### macOS
```bash
# GUI app (recommended)
brew install --cask neovide-app

# CLI binary
brew install neovide
```

### Linux
```bash
# Snap
sudo snap install neovide

# Arch Linux (extra repo)
sudo pacman -S neovide
```

### Windows
```bash
# Scoop
scoop bucket add extras
scoop install neovide
```

Or download the installer from the [GitHub Releases page](https://github.com/neovide/neovide/releases).

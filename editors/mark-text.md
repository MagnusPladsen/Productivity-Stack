# Mark Text

> Free, open-source WYSIWYG Markdown editor with real-time preview, focus mode, and support for GitHub Flavored Markdown — a Typora alternative.

**Platforms:** macOS, Linux, Windows

> **Note:** The Homebrew cask is deprecated as of August 2025 and scheduled for removal September 2026. On Apple Silicon, run `xattr -cr /Applications/MarkText.app` if you see a "damaged" warning.

## Links
- [GitHub Repository](https://github.com/marktext/marktext)
- [Official Website](https://www.marktext.cc)

## Install

### macOS
```bash
brew install --cask mark-text
```

Or download the `.dmg` from the [GitHub Releases page](https://github.com/marktext/marktext/releases).

### Linux

**Snap:**
```bash
sudo snap install marktext
```

Or download the `.AppImage` from the [GitHub Releases page](https://github.com/marktext/marktext/releases).

### Windows
```bash
winget install -e --id MarkText.MarkText
```

Or download the setup `.exe` from the [GitHub Releases page](https://github.com/marktext/marktext/releases).

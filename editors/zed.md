# Zed

> High-performance multiplayer code editor built in Rust, from the creators of Atom and Tree-sitter.

**Platforms:** macOS, Linux

## Links
- [Official Website](https://zed.dev/)
- [GitHub Repository](https://github.com/zed-industries/zed) ⭐ 76.9k
- [Documentation](https://zed.dev/docs/)

## Install

### macOS
```bash
brew install --cask zed
```

Or download the `.dmg` from the [download page](https://zed.dev/download), open it, and drag Zed to your Applications folder.

### Linux
```bash
curl -f https://zed.dev/install.sh | sh
```

Supports Ubuntu, Arch, Debian, and other major distributions.

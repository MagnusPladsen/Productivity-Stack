# Lapce

> Lightning-fast code editor written in Rust with GPU rendering, built-in LSP, and modal editing support.

**Platforms:** macOS, Linux, Windows

## Links
- [Official Website](https://lapce.dev)
- [GitHub Repository](https://github.com/lapce/lapce) ⭐ 38.1k
- [Documentation](https://docs.lapce.dev)

## Install

### macOS
```bash
brew install --cask lapce
```

Or download the `.dmg` from the [GitHub Releases page](https://github.com/lapce/lapce/releases).

### Linux

**Flatpak (all distros):**
```bash
flatpak install flathub dev.lapce.lapce
```

Or download the pre-built binary from the [GitHub Releases page](https://github.com/lapce/lapce/releases).

**Snap:**
```bash
sudo snap install lapce
```

### Windows

Download the installer from the [GitHub Releases page](https://github.com/lapce/lapce/releases) and run it.

# Helix

> A post-modern modal text editor built in Rust with built-in LSP, tree-sitter syntax highlighting, and multiple selections — no plugins required.

**Platforms:** macOS, Linux, Windows

## Links
- [Official Website](https://helix-editor.com)
- [GitHub Repository](https://github.com/helix-editor/helix)
- [Documentation](https://docs.helix-editor.com)

## Install

### macOS
```bash
brew install helix
```

### Linux

**Debian/Ubuntu:**
```bash
sudo apt install helix
```

**Arch Linux:**
```bash
sudo pacman -S helix
```

Or download the pre-built binary or AppImage from the [GitHub Releases page](https://github.com/helix-editor/helix/releases).

### Windows
```bash
winget install Helix.Helix
```

Or with Scoop:
```bash
scoop install helix
```

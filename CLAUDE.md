# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Overview

This is a documentation-only repository containing a curated list of free productivity tools and programs. The entire content lives in a single `README.md` file — there is no application code, build system, or tests.

## Structure

- `README.md` — The complete tool list, organized by category (Terminals, Terminal Tools, Editors, Note Taking, Development Tools, Design Tools, AI Tools, Package Managers, Productivity Tools, Image Editing, Browsers)

## Conventions

- Each tool entry follows the format: bold name, one-line description, link emoji + URL
- Categories are separated by horizontal rules (`* * *`)
- Table of Contents at the top links to each tool via anchor tags
- New tools should be added in the appropriate category with the same format
- Contributing workflow: fork, add tool, submit PR

# Kap

> Lightweight macOS screen recorder that exports to GIF, MP4, WebM, and APNG.

**Platforms:** macOS

## Links
- [Official Website](https://getkap.co)
- [GitHub Repository](https://github.com/wulkano/Kap)

## Install

### macOS
```bash
brew install --cask kap
```

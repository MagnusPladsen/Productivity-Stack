# Notion

> All-in-one workspace for notes, wikis, databases, and project management.

**Platforms:** macOS, Windows

## Links
- [Official Website](https://www.notion.so/)
- [Desktop App](https://www.notion.com/desktop)

## Install

### macOS
```bash
brew install --cask notion
```

### Windows
```bash
winget install Notion.Notion
```

> **Linux:** Notion does not publish an official Linux desktop app. Community-maintained wrappers are available on the [Snap Store](https://snapcraft.io/notion-desktop), but these are unofficial.

# Joplin

> Open-source note-taking and to-do application with Markdown support and end-to-end encrypted sync.

**Platforms:** macOS, Linux, Windows

## Links
- [Official Website](https://joplinapp.org/)
- [GitHub Repository](https://github.com/laurent22/joplin)

## Install

### macOS
```bash
brew install --cask joplin
```

### Linux
```bash
wget -O - https://raw.githubusercontent.com/laurent22/joplin/dev/Joplin_install_and_update.sh | bash
```

### Windows
```bash
winget install --id Joplin.Joplin -e
```

# Obsidian

> A powerful knowledge base and note-taking app that works on top of a local folder of plain text Markdown files.

**Platforms:** macOS, Linux, Windows

## Links
- [Official Website](https://obsidian.md/)
- [Help & Download](https://help.obsidian.md/install)

## Install

### macOS
```bash
brew install --cask obsidian
```

### Linux
Download the `.AppImage` or `.deb` from the [official download page](https://obsidian.md/download).

### Windows
```bash
winget install Obsidian.Obsidian
```

# GIMP

> A free and open-source image editor for photo retouching, image composition, and image authoring.

**Platforms:** macOS, Linux, Windows

## Links
- [Official Website](https://www.gimp.org/)
- [Downloads](https://www.gimp.org/downloads/)
- [GitHub Repository](https://github.com/GNOME/gimp)

## Install

### macOS

```bash
brew install --cask gimp
```

### Linux

Install via Flatpak (recommended for the latest version):

```bash
flatpak install flathub org.gimp.GIMP
```

Or download the AppImage from the [official downloads page](https://www.gimp.org/downloads/).

### Windows

```bash
winget install GIMP.GIMP
```

Or download the installer from the [official downloads page](https://www.gimp.org/downloads/).

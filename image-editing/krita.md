# Krita

> A free and open-source digital painting application designed for concept artists, illustrators, and VFX artists.

**Platforms:** macOS, Linux, Windows

## Links
- [Official Website](https://krita.org/)
- [Downloads](https://krita.org/en/download/)
- [GitHub Repository](https://github.com/KDE/krita) ⭐ 9.4k

## Install

### macOS

```bash
brew install --cask krita
```

### Linux

Install via Flatpak:

```bash
flatpak install flathub org.kde.krita
```

Or download the AppImage from the [official downloads page](https://krita.org/en/download/).

### Windows

```bash
winget install KDE.Krita
```

Or download the installer from the [official downloads page](https://krita.org/en/download/).

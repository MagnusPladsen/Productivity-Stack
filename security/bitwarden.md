# Bitwarden

> Open-source password manager with free cloud sync across unlimited devices.

**Platforms:** macOS, Linux, Windows

> **Free tier:** Unlimited passwords, unlimited devices, browser extensions, mobile apps.
> **Paid ($10/year):** TOTP authenticator, encrypted file attachments, emergency access, priority support.

## Links
- [Official Website](https://bitwarden.com)
- [GitHub Repository](https://github.com/bitwarden/clients) ⭐ 12.5k

## Install

### macOS
```bash
brew install --cask bitwarden
```

### Linux
```bash
# Snap
sudo snap install bw

# Flatpak
flatpak install flathub com.bitwarden.desktop

# CLI only (npm)
npm install -g @bitwarden/cli
```

### Windows
```bash
winget install -e --id Bitwarden.Bitwarden
```

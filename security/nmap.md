# nmap

> Network scanner and security auditing tool for discovering hosts, open ports, and services.

**Platforms:** macOS, Linux, Windows

## Links
- [Official Website](https://nmap.org)
- [GitHub Repository](https://github.com/nmap/nmap)

## Install

### macOS
```bash
brew install nmap
```

### Linux
```bash
# Ubuntu/Debian
sudo apt install nmap

# Fedora/RHEL
sudo dnf install nmap
```

### Windows
```bash
winget install -e --id Insecure.Nmap
```

## Alternatives
- [Wireshark](wireshark.md) — passive packet capture and protocol analysis

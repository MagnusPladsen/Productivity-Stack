# Wireshark

> Network protocol analyzer for capturing and inspecting network traffic in real time.

**Platforms:** macOS, Linux, Windows

## Links
- [Official Website](https://wireshark.org)
- [GitHub Repository](https://github.com/wireshark/wireshark) ⭐ 9.1k

## Install

### macOS
```bash
brew install --cask wireshark
```

### Linux
```bash
# Ubuntu/Debian (official PPA for latest version)
sudo add-apt-repository ppa:wireshark-dev/stable
sudo apt update
sudo apt install wireshark
```

### Windows
```bash
winget install -e --id WiresharkFoundation.Wireshark
```

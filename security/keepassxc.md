# KeePassXC

> Offline password manager. No cloud, no account — your database stays on your device.

**Platforms:** macOS, Linux, Windows

## Links
- [Official Website](https://keepassxc.org)
- [GitHub Repository](https://github.com/keepassxreboot/keepassxc) ⭐ 26.2k

## Install

### macOS
```bash
brew install --cask keepassxc
```

### Linux
```bash
# Flatpak (recommended)
flatpak install flathub org.keepassxc.KeePassXC

# Ubuntu/Debian (may lag behind latest release)
sudo apt install keepassxc
```

### Windows
```bash
winget install -e --id KeePassXCTeam.KeePassXC
```

# Security

Tools for local HTTPS, password management, and network analysis.

| Tool | Platforms | Open Source | Description |
|------|-----------|-------------|-------------|
| [mkcert](mkcert.md) | macOS, Linux, Windows | Yes | Zero-config local HTTPS certificates |
| [Bitwarden](bitwarden.md) | macOS, Linux, Windows | Yes | Cloud-synced password manager (free tier available) |
| [KeePassXC](keepassxc.md) | macOS, Linux, Windows | Yes | Offline password manager, no cloud required |
| [Wireshark](wireshark.md) | macOS, Linux, Windows | Yes | Network protocol analyzer |
| [nmap](nmap.md) | macOS, Linux, Windows | Yes | Network scanner and security auditing |
